# How to Create an Issue

Each script in the archive should have its own Issue to track its progress.

## Steps
1. Go to the **Issues** tab in the repo
2. Click **New Issue**
3. Select the **Script Submission** template
4. Fill in the following:
   - Script title in the issue title field e.g. `[SCRIPT] NumPy Introduction`
   - Topic, author, and Google Drive link in the body
5. Apply the appropriate label (see label guide)
6. Click **Submit new issue**
