# How to Use Labels

Labels track where a script is in the workflow.
Only one status label should be applied to an issue at a time.

## Labels

| Label | When to Apply |
|---|---|
| `archive-management` | General repo organisation and maintenance |
| `script-draft` | Script is currently being written |
| `script-finalised` | Script has been reviewed and approved |
| `queued-for-ig` | Script is ready to be adapted for Instagram |
| `posted-to-ig` | Instagram post has been published |

## How to Update a Label
1. Open the Issue
2. Click the **Labels** section on the right panel
3. Deselect the old label and select the new one
4. Changes save automatically
