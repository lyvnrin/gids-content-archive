# How to Upload a Script

1. Convert your Google Doc to a `.md` or `.txt` file
   - File → Download → Plain Text (.txt) or save as Markdown
2. Rename the file clearly e.g. `numpy-intro.md`, `ada-lovelace.md`
3. Navigate to the correct folder inside `/post-topics/`
4. Click **Add file → Upload files**
5. Drag and drop your file in
6. Write a short commit message e.g. `Add NumPy intro script`
7. Click **Commit changes**

## Naming Conventions
- Use lowercase and hyphens only e.g. `data-cleaning-basics.md`
- No spaces, no special characters
- Keep names short but descriptive
