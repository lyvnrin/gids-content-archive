# Girls in Data Science | Content Archive

A centralised archive of all long-form technical scripts produced by GIDS associates.
This repo is the home base for content, from first draft to Instagram publication.

---

## 📁 Structure

- [`/guides`](./guides/) - Start here if you're new. How to use the repo, create issues, upload scripts and more.
- [`/post-topics`](./post-topics/) - All scripts organised by topic (Python, Neurology, Famous Figures, Data Science Resources)
- [`ig-adapted.md`](./ig-adapted.md) - A running list of all published Instagram post links

---

## 🏷️ Status Labels

| Label | Meaning |
|---|---|
| `script-draft` | Script is currently being written |
| `script-finalised` | Script has been reviewed and approved |
| `queued-for-ig` | Script is ready to be adapted for Instagram |
| `posted-to-ig` | Instagram post has been published |

---

## 📌 Quick Start

1. Read the [`/guides`](./guides/workflow-overview.md) folder before contributing
2. Upload your script to the correct `/post-topics` folder, or create a new folder for your topic
3. Open an Issue
4. Apply the relevant label and track progress on the Project board
